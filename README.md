#   Lyrics Finder
This project uses the [lyrics.ovh]( https://lyricsovh.docs.apiary.io/#reference/0/lyrics-of-a-song) Public API to get and display song lyrics from a title and song tile chosen by the user.

## Usage
### Clone the Repository
In the shell:
- git clone https://github.com/ryanyonek/Lyrics-Finder.git
- cd Lyrics-Finder

### Node
This project uses Node.
- Install: https://nodejs.org/en/download

### Run
In the shell:
- npm i
- node index.js
